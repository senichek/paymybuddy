Pay My Buddy

This is one of the projects that I did during my training at Openclassrooms. In this project I used Spring Boot, Spring Security, Spring Data JPA, Hibernate, PostgreSQL, JUnit, Bootstrap.

Description of the course

The features of the project:

1. Registration of a user;
2. User can add another user to his "connections" (friend's list);
3. User can transfer money to his friends;
4. User can add balance (money) to his/her own account;
5. User can modify his/her personal information;
